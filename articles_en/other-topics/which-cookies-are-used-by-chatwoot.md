# Which cookies are used by Katalis?Katalis uses the following cookies.

Chat Widget​

cw_conversation: This cookie lets Katalis persist the conversation when the contact navigates through the webpages or
revisits the website later.

cw_user_{identifier}: This cookie is used to cache the contact details if a contact is identified via the setUser SDK
method.

Agent Dashboard​

_katalis_session: Used by the Katalis Super Admin panel to persist the user session.

cw_d_session_info: Used by the Agent Dashboard to store the logged-in user's session.Last updated on Nov 09, 2023
