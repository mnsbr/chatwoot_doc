# How to migrate from Front to Katalis.app?

If you're currently using Front and want to switch to Katalis.app, this guide will walk you through the step-by-step

process of migrating your conversations, contacts, live chat widget, and setting up Katalis.app. This is a very quick
process; let’s dive in.

## Before you begin

Before setting up Katalis.app, there are two essential preliminary tasks to complete in order to smoothly remove Front from
your ecosystem.

## Step 1. Remove the Front chat widget

To start, please make sure to uninstall the existing live chat code that you would have previously integrated for
enabling the Front live chat widget on your website.

## Step 2. Export contacts

Next, export all of your contacts from Front, with the help of this guide.

## Migrating to Katalis.app

## Step 3. Sign up

Create an account in Katalis.app, using this guide.

## Step 4. Migrate the live chat widget

Create a live chat inbox in Katalis.app, using this guide. At the end of the setup, you will find the live chat code
snippet to be pasted into your website’s codebase. Complete this step, ensuring that you have already removed the live
chat code snippet for Front’s live chat widget, as mentioned in Step 1.

## Step 5. Migrate contacts

Open the ‘Contacts’ tab in Katalis.app, press the Import button.

Next, follow the on-screen instructions to upload the Contacts file exported from Front, in a csv format. Once your
upload is completed, all of your contacts will be successfully synced in Katalis.app.

## Step 6. Migrate conversations

For importing your existing conversations from Front to Katalis.app, use this API.
