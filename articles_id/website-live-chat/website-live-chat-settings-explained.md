# Pengaturan Live Chat Website Dijelaskan

Dokumen ini akan membantu Anda menyiapkan live chat website secara lengkap. Dokumen ini menjelaskan cara membuat dan menggunakan pengaturan tambahan untuk inbox website yang diinginkan.

## Kompatibilitas dan Persyaratan

Dashboard web dan widget live chat berfungsi dengan sebagian besar browser web modern. Berikut versi browser yang didukung:

- Mozilla Firefox: v52 ke atas
- Google Chrome: v57 ke atas
- Chromium: v57 ke atas
- Apple Safari: v14.1 ke atas
- Microsoft Edge: v16 ke atas

Versi OS mobile yang didukung untuk aplikasi mobile Katalis.app:

- Android: v5.0 ke atas
- iOS: v11.0 ke atas

## Cara Membuat Channel Website di Katalis.app?

**Langkah 1.** Buka halaman Inboxes. Buka Settings → Inboxes → tombol "Add Inbox".

**Langkah 2.** Klik ikon "Website".

**Langkah 3.** Masukkan nama website, URL, dan detail lainnya seperti yang dijelaskan di bawah.

Kolom yang tersedia:

1. **Nama website** - Masukkan nama website Anda. Contoh: Example Inc.
2. **Domain website** - Masukkan URL website Anda. Contoh: www.example.com
3. **Warna widget** - Klik palet dan pilih warna untuk widget live chat website Anda.
4. **Heading selamat datang** - Tulis pesan sapaan untuk pelanggan. Contoh: Halo 👋
5. **Tagline selamat datang** - Contoh: Kami di sini untuk membantu pertanyaan Anda. Tanyakan apa saja, atau bagikan masukan Anda.
6. **Aktifkan/nonaktifkan salam channel** - Aktifkan untuk mengirim pesan salam saat pelanggan memulai percakapan.
7. **Pesan salam channel** - Berikan salam untuk memulai percakapan. Contoh: Hai. Ada yang bisa kami bantu?

Setelah semua detail dimasukkan, klik tombol "Create Inbox".

**Langkah 4.** "Add agents" ke inbox website Anda.

Percakapan hanya akan diarahkan ke agen jika mereka terkait dengan inbox. Bahkan jika Anda adalah Administrator, Anda perlu menambahkan diri sendiri sebagai agen ke Inbox untuk melihat Inbox di layar Anda.

Pilih Agen dari dropdown dan klik Add agents. Jika agen belum ditambahkan, ikuti langkah untuk menambahkan agen.

**Langkah 5.** Hore! Anda telah berhasil membuat inbox website. Sekarang, tambahkan widget ke website Anda.

Salin dan tempel cuplikan kode yang ditampilkan di halaman ke website Anda dan mulai mendukung pelanggan. Anda bisa mengklik tombol Copy untuk menyalin dan kemudian menempelkan di file root website Anda.

**Langkah 6.** Jika ingin memperbarui agen dengan akses ke inbox, buka Settings → Inboxes. Anda akan melihat inbox website terdaftar di sana. Klik Settings. Anda akan melihat kode serta daftar agen yang memiliki akses ke inbox.

## Menyelesaikan Pengaturan Inbox Website

Anda sudah memiliki inbox website yang fungsional. Mari lihat pengaturan Inbox tambahan.

### Tab Settings

Gulir ke bawah untuk melihat lebih banyak pengaturan, di mana Anda bisa menyiapkan berbagai notifikasi yang ingin diterima.

- **Enable email collect box** diaktifkan secara default. Ini berarti pesan otomatis untuk mengumpulkan ID email akan ditampilkan saat pelanggan memulai percakapan.
- **Enable CSAT.** Setelah diaktifkan, survei Kepuasan Pelanggan akan diluncurkan setiap kali tiket diselesaikan. Hasil CSAT bisa dilihat nanti di bagian Reports.

Klik tombol "Update" untuk menyimpan perubahan.

### Tab Collaborators

Di sini, Anda bisa menambahkan Agen ke Inbox atau menghapus Agen yang sudah ditambahkan. Anda juga bisa mengaktifkan atau menonaktifkan penugasan otomatis percakapan baru ke agen.

Klik Update untuk menyimpan perubahan.

### Tab Business Hours

Di bagian ini, Anda bisa mengatur jam kerja tim Anda.

Centang kotak Enable business availability for this inbox jika Anda memiliki jam kerja yang terdefinisi. Setelah dicentang, bagian baru akan terbuka. Anda bisa menulis pesan yang ditampilkan ke pelanggan selama di LUAR jam kerja.

**Catatan:** Ketika Captain diaktifkan di inbox, pesan ketidaktersediaan otomatis ditekan. Captain akan menangani percakapan secara langsung, bahkan di luar jam kerja.

Anda juga bisa memilih jam kerja yang tersedia untuk setiap hari dalam seminggu, beserta zona waktu yang tepat.

### Tab Pre Chat Form

Dengan ini, Anda bisa mengumpulkan informasi tentang pelanggan segera setelah mereka masuk ke jendela live chat/widget website.

Kolom yang tersedia:

1. **Enable pre chat form** - Pilih Ya atau Tidak.
2. **Pre Chat Message** - Tulis pesan untuk memulai percakapan. Contoh: Hai, Kami membutuhkan beberapa informasi untuk melayani Anda lebih baik.
3. **Pre chat form fields** - Centang yang diperlukan. **Penting:** "Name," "Email" dan "Phone" adalah satu-satunya atribut standar untuk formulir pre-chat. Kolom lain dibuat melalui atribut kustom.

Klik tombol "Update". Formulir Pre Chat sekarang aktif.

### Tab Configuration

Anda bisa menemukan cuplikan kode yang harus disalin ke website di sini.

### Tab Widget Builder

Anda bisa menyesuaikan widget web dengan pengaturan sederhana di halaman ini. Anda akan bisa melihat perubahan secara live.
