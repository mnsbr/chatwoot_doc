# Cara Menginstal Live Chat di Website WordPress

WordPress adalah Sistem Manajemen Konten yang populer. Jika Anda menghosting website di WordPress, Anda bisa menambahkan widget live chat Katalis dan berbicara dengan pengunjung secara real time. Ini bisa dilakukan dalam 4 langkah sederhana menggunakan plugin WordPress Katalis.

**Langkah 1.** Unduh plugin Katalis

Unduh file Zip plugin WordPress Katalis terbaru dari GitHub, seperti yang ditunjukkan di bawah.

**Langkah 2.** Unggah plugin ke website WordPress Anda

Login ke akun WordPress Anda. Navigasi ke Admin Panel dan klik "Plugins" dari sidebar. Anda akan melihat tombol "Add New".

Anda akan melihat opsi untuk mengunggah plugin. Klik "Upload" dan pilih file Zip yang diunduh di langkah 1.

**Langkah 3.** Aktifkan plugin

Setelah plugin diunggah, Anda akan melihat layar Sukses dengan opsi "Activate Plugin". Klik tombol "Activate Plugin".

Selamat! Plugin sekarang terinstal. Untuk menyelesaikan konfigurasi, Anda harus membagikan kredensial aplikasi Katalis.

**Langkah 4.** Konfigurasi plugin

Sekarang, Anda akan bisa melihat "Katalis Settings" di bawah menu "Settings". Klik untuk memperbarui konfigurasi live chat.

Anda akan disajikan layar Settings. Konfigurasi plugin Katalis di sini agar berfungsi.

Anda perlu membuat inbox Website di instalasi Katalis Anda. Silakan lihat panduan ini untuk membuat inbox channel website.

Kolom formulir dijelaskan di bawah:

1. **Token Website Katalis / URL Instalasi Katalis** - Setelah membuat inbox channel website di instalasi Katalis, Anda akan mendapat skrip. Anda bisa menyalin Token dan URL.

2. **Desain Widget** - Katalis mendukung dua desain widget live chat – Desain Standar dan desain Expanded Bubble. Pilih preferensi Anda.

3. **Posisi Widget** - Anda bisa memilih di mana menempatkan widget live chat di website berdasarkan preferensi. Opsi yang tersedia adalah Kiri & Kanan.

4. **Bahasa** - Katalis mendukung 30+ bahasa. Bahasa default adalah English.

5. **Teks Launcher** - Ini adalah kolom opsional. Jika Anda menggunakan desain expanded bubble, Anda bisa menggunakan teks launcher untuk menyesuaikan teks yang ditampilkan di bubble.

Setelah mengisi formulir, klik tombol "Save changes". Pastikan juga bahwa Anda telah menyiapkan inbox website dari Katalis.

**Langkah 5.** Verifikasi bahwa plugin berfungsi dengan benar

Kunjungi website Anda dan lihat apakah widget muncul di website!
