# Cara Mengaktifkan Push Notification di Browser Anda

## Google Chrome:

1. Klik menu tiga titik di pojok kanan atas dan pilih Settings.
2. Navigasi ke Privacy and security > Site Settings > Notifications.
3. Untuk memastikan notifikasi bisa diaktifkan, centang "Sites can ask to send notifications".
4. Untuk mengelola situs tertentu, gunakan tombol Add di bawah Allow atau Block untuk memasukkan URL situs.

Alternatifnya, klik ikon gembok di address bar saat berada di situs, pilih Site settings, dan sesuaikan izin Notifications.

## Mozilla Firefox:

1. Klik tombol menu (tiga garis horizontal) dan pilih Settings.
2. Buka Privacy & Security.
3. Di bawah Permissions, temukan Notifications dan klik Settings.
4. Di sini, Anda bisa mengelola izin untuk setiap situs atau hapus centang "Block new requests asking to allow notifications" untuk memastikan prompt ditampilkan.

## Safari (macOS):

1. Klik Safari di menu atas dan pilih Preferences.
2. Buka tab Websites dan pilih Notifications.
3. Di sini, Anda bisa mengizinkan atau menolak izin notifikasi untuk setiap website yang terdaftar. Pastikan "Allow websites to ask for permissions" diaktifkan.

## Microsoft Edge:

1. Klik menu tiga titik di pojok kanan atas dan pilih Settings.
2. Navigasi ke Cookies and site permissions > Notifications.
3. Untuk memblokir semua notifikasi, nonaktifkan "Ask before sending".
4. Untuk mengelola situs tertentu, gunakan tombol Add di bawah Allow atau Block untuk memasukkan URL situs.

Alternatifnya, klik ikon gembok di address bar saat berada di situs, pilih Permissions for this site, dan sesuaikan pengaturan Notifications.
