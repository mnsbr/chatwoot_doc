# Cookie Apa yang Digunakan oleh Katalis.app?

Katalis.app menggunakan cookie berikut.

## Widget Chat

**cw_conversation:** Cookie ini memungkinkan Katalis.app mempertahankan percakapan saat kontak menavigasi halaman web atau mengunjungi kembali website nanti.

**cw_user_{identifier}:** Cookie ini digunakan untuk menyimpan cache detail kontak jika kontak diidentifikasi melalui metode SDK setUser.

## Dashboard Agen

**_Katalis.app_session:** Digunakan oleh panel Super Admin Katalis.app untuk mempertahankan sesi pengguna.
