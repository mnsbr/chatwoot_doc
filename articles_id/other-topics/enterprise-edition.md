# Enterprise Edition

Mulai dari versi 2.0, Katalis.app mengirimkan dua versi softwarenya. Edisi community berlisensi MIT, dan edisi enterprise tersedia di bawah lisensi proprietary. Kedua versi software dibangun dari satu repositori GitHub.

Sementara edisi community akan selalu tetap gratis dan open-source, edisi enterprise adalah upaya untuk menopang kedua penawaran dan Katalis.app sebagai bisnis komersial. Model ini diambil dari perusahaan open-source komersial lain seperti Gitlab, Mattermost, Metabase, dll.

## Perbedaan antara Edisi Community dan Enterprise

Edisi community Katalis.app memiliki semua fitur komunikasi pelanggan yang relevan yang dibutuhkan bisnis yang berkembang. Pada saat yang sama, edisi enterprise berisi fitur proprietary seperti dashboard yang bisa dikustomisasi, SLA Management, software penjadwalan agen, IP blocklisting, dll., yang cocok untuk organisasi yang lebih besar.

Sementara source code untuk kedua edisi terbuka untuk publik di repositori GitHub kami, semua konten folder /enterprise dilindungi hak cipta.

## Memilih Edisi yang Tepat

Kami merekomendasikan Anda mengunduh dan menginstall Katalis.app Enterprise Edition agar bisa upgrade ke tier berbayar di masa depan tanpa harus menginstall ulang Katalis.app.

### Enterprise Edition

Edisi Enterprise menawarkan hal berikut:

- Semua fitur Katalis.app
- Source code lengkap yang terlihat dan bisa diaudit
- Semua fitur edisi community tersedia tanpa berlangganan
- Opsi Dukungan Langsung
- Opsi untuk upgrade dengan mudah ke fitur berbayar

### Community Edition

Edisi Community menawarkan hal berikut:

- Source code lengkap yang terlihat dan bisa dimodifikasi
- Tidak ada akses ke fitur berbayar
- Hanya Community Support
- Akan memerlukan migrasi ke edisi Enterprise untuk menambahkan fitur berbayar di masa depan
