# Aplikasi Mobile untuk Android

Anda bisa mengakses akun Katalis.app melalui aplikasi mobile Katalis.app di perangkat Android.

## Cara Mendapatkan Aplikasi Mobile Katalis.app untuk Android

Anda bisa mengunduh aplikasi bernama "Katalis.app" dari Play Store. Ikuti langkah-langkah berikut:

1. Buka Play Store di perangkat Android Anda yang menjalankan versi 7.0 atau lebih baru.
2. Cari Katalis.app dan tekan "Install" untuk mengunduh aplikasi.
3. Setelah diunduh, buka aplikasi Katalis.app.
4. Masukkan Installation URL; dalam format domain.com. Jika Anda menggunakan instalasi Katalis.app self-hosted, masukkan URL server Anda. Jika tidak, gunakan app.katalis.app.
5. Klik tombol Connect dan masukkan kredensial akun Anda untuk login.

## Berkontribusi dalam Pengujian Pembaruan Aplikasi

Anda bisa mendaftar untuk program beta kami untuk membantu menguji pembaruan aplikasi sebelum rilis. Buka link ini dari perangkat Android Anda.

Anda bisa keluar dari program Beta Testing kapan saja dengan mengklik link ini saat masuk dengan alamat email Google Play yang digunakan untuk mendaftar program Beta. Di link tersebut, klik "Leave the program".
