# Aplikasi Mobile untuk iOS

Anda bisa mengakses akun Katalis.app di perangkat mobile iOS melalui aplikasi mobile Katalis.app.

## Cara Mendapatkan Aplikasi Mobile Katalis.app untuk iOS

Anda bisa mengunduh aplikasi bernama "Katalis.app" dari App Store. Ikuti langkah-langkah berikut:

1. Buka App Store di perangkat Apple Anda yang menjalankan iOS 11.0 atau lebih baru.
2. Cari Katalis.app dan tekan "GET" untuk mengunduh aplikasi.
3. Setelah diunduh, buka aplikasi Katalis.app.
4. Masukkan Installation URL; dalam format domain.com. Jika Anda menggunakan instalasi Katalis.app self-hosted, masukkan URL server Anda. Jika tidak, gunakan app.katalis.app.
5. Klik tombol Connect dan masukkan kredensial akun Anda untuk login.

## Berkontribusi dalam Pengujian Pembaruan Aplikasi

Anda bisa mendaftar untuk program beta kami untuk membantu menguji pembaruan aplikasi sebelum rilis. Buka link ini dari perangkat iOS Anda.

Anda bisa keluar dari program Beta Testing kapan saja dengan membuka aplikasi TestFlight dan mengklik "Stop Testing".
