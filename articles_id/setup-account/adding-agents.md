# Menambahkan Agen

Agen adalah anggota tim Komunikasi Pelanggan Anda di Katalis. Agen akan dapat melihat dan membalas pesan dari pengguna Anda.

Untuk menambahkan agen ke akun Katalis Anda, Anda bisa menggunakan tautan 'Klik di sini untuk mengundang anggota tim', seperti yang ditandai di tangkapan layar di bawah.

Atau, Anda bisa navigasi dari Settings → Agents.

Anda akan diarahkan ke halaman manajemen Agen seperti yang ditunjukkan di bawah. Nama Anda akan terdaftar secara default sebagai Administrator. Administrator bisa mengakses semua fitur Katalis yang diaktifkan untuk akun Anda, termasuk Pengaturan.

Klik tombol Add Agent di sisi kanan atas.

Jendela pop-up akan muncul di mana Anda bisa memasukkan detail Agen seperti nama, peran, dan alamat email.

Klik tombol Add Agent untuk mengonfirmasi. Anda akan melihat pesan Agent Added Successfully. Agen ini sekarang akan terdaftar di halaman manajemen Agen Anda.

## Cara Mengubah atau Menghapus Agen?

**Langkah 1.** Buka daftar agen Anda dari Settings → Agents. Temukan agen yang ingin Anda edit. Klik tombol Edit (ikon pensil) jika ingin mengeditnya. Klik tombol hapus (ikon silang merah) jika ingin menghapusnya.

**Langkah 2.** Jika Anda mengedit info agen, Anda akan melihat modal dengan informasi yang sudah terisi. Anda bisa mengedit nama Agen dan perannya. Klik Edit Agent untuk menyimpan perubahan. Klik Cancel jika ingin membatalkan perubahan.

**Selanjutnya: Menambahkan inbox**
