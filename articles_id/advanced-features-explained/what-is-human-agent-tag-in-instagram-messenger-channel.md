
# Apa itu Human Agent Tag di Channel Instagram/Messenger?

Fitur Human Agent memungkinkan bisnis untuk merespons pesan pengguna di luar jendela pengiriman pesan standar 24 jam. Ketika agen manusia menerapkan tag pesan `human_agent`, mereka bisa membalas pesan pengguna dalam 7 hari dari pesan terakhir pengguna.

Fitur ini sangat berguna dalam skenario di mana masalah membutuhkan lebih banyak waktu untuk penyelesaian. Kasus penggunaan umum meliputi:

- Ketika bisnis tutup (misalnya akhir pekan atau hari libur)
- Pertanyaan pelanggan yang kompleks yang memerlukan lebih dari 24 jam untuk diselesaikan

Meskipun izin HUMAN_AGENT sudah diaktifkan untuk instance cloud kami, instalasi self-hosted harus menjalani dan melewati proses App Review untuk mengakses fitur ini.
