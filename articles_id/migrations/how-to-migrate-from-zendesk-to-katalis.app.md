# Cara Migrasi dari Zendesk ke Katalis.app

Jika Anda saat ini menggunakan Zendesk dan ingin beralih ke Katalis.app, panduan ini akan memandu Anda melalui proses langkah demi langkah untuk memigrasikan percakapan, kontak, widget live chat, dan menyiapkan Katalis.app. Ini adalah proses yang sangat cepat; mari mulai.

## Sebelum Memulai

Sebelum menyiapkan Katalis.app, ada dua tugas penting yang harus diselesaikan agar bisa menghapus Zendesk dari ekosistem Anda dengan lancar.

**Langkah 1. Hapus widget chat Zendesk**

Untuk memulai, pastikan untuk menghapus kode live chat yang sebelumnya diintegrasikan untuk mengaktifkan widget live chat Zendesk di website Anda.

**Langkah 2. Ekspor kontak**

Selanjutnya, ekspor semua kontak dari Zendesk, dengan bantuan panduan ini.

## Migrasi ke Katalis.app

**Langkah 3. Daftar**

Buat akun di Katalis.app, menggunakan panduan ini.

**Langkah 4. Migrasikan widget live chat**

Buat inbox live chat di Katalis.app, menggunakan panduan ini. Di akhir pengaturan, Anda akan menemukan snippet kode live chat yang perlu ditempel ke codebase website Anda. Selesaikan langkah ini, pastikan Anda sudah menghapus snippet kode live chat Zendesk, seperti disebutkan di Langkah 1.

**Langkah 5. Migrasikan kontak**

Buka tab 'Contacts' di Katalis.app, tekan tombol Import.

Selanjutnya, ikuti instruksi di layar untuk mengunggah file Contacts yang diekspor dari Zendesk, dalam format CSV. Setelah unggahan selesai, semua kontak Anda akan berhasil tersinkronisasi di Katalis.app.

**Langkah 6. Migrasikan percakapan**

Untuk mengimpor percakapan yang ada dari Zendesk ke Katalis.app, gunakan API ini.
