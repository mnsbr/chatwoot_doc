# Cara Migrasi dari Intercom ke Katalis

Jika Anda saat ini menggunakan Intercom dan ingin beralih ke Katalis, panduan ini akan memandu Anda melalui proses langkah demi langkah untuk memigrasikan percakapan, kontak, widget live chat, dan menyiapkan Katalis. Ini adalah proses yang sangat cepat; mari mulai.

## Sebelum Memulai

Sebelum menyiapkan Katalis, ada dua tugas penting yang harus diselesaikan agar bisa menghapus Intercom dari ekosistem Anda dengan lancar.

**Langkah 1. Hapus widget chat Intercom**

Untuk memulai, pastikan untuk menghapus kode live chat yang sebelumnya diintegrasikan untuk mengaktifkan widget live chat Intercom di website Anda.

**Langkah 2. Ekspor kontak**

Selanjutnya, kunjungi tab 'Contacts' di akun Intercom Anda. Buka 'All users' → buka dropdown 'More' → 'Export users'.

## Migrasi ke Katalis

**Langkah 3. Daftar**

Buat akun di Katalis, menggunakan panduan ini.

**Langkah 4. Migrasikan widget live chat**

Buat inbox live chat di Katalis, menggunakan panduan ini. Di akhir pengaturan, Anda akan menemukan snippet kode live chat yang perlu ditempel ke codebase website Anda. Selesaikan langkah ini, pastikan Anda sudah menghapus snippet kode live chat Intercom, seperti disebutkan di Langkah 1.

**Langkah 5. Migrasikan kontak**

Buka tab 'Contacts' di Katalis, tekan tombol Import.

Selanjutnya, ikuti instruksi di layar untuk mengunggah file Contacts yang diekspor dari Intercom, dalam format CSV. Setelah unggahan selesai, semua kontak Anda akan berhasil tersinkronisasi di Katalis.

**Langkah 6. Migrasikan percakapan**

Untuk mengimpor percakapan yang ada dari Intercom ke Katalis, gunakan API ini.
