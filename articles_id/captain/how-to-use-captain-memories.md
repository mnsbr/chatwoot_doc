
# Cara Menggunakan Captain Memories

Captain Memories adalah potongan informasi yang disimpan dari percakapan pelanggan untuk memberikan konteks bagi interaksi di masa depan. Memories ini memungkinkan respons yang dipersonalisasi, efisien, dan sadar konteks.

Misalnya, jika pelanggan menyebutkan bahwa mereka memiliki tim beranggotakan 18 orang atau membagikan preferensi tertentu, Captain akan menyimpan informasi ini. Ini memungkinkan tim Anda menggunakan wawasan ini selama percakapan di masa depan untuk meningkatkan pengalaman pelanggan.

## Jenis Informasi Apa yang Disimpan Captain?

Captain dirancang untuk menyimpan detail yang dibagikan pelanggan yang relevan untuk memberikan dukungan yang dipersonalisasi. Ini mungkin termasuk:

- Preferensi pelanggan (misalnya, gaya komunikasi yang disukai atau preferensi produk).
- Pertanyaan dan masalah berulang (misalnya, "Bagaimana cara mengelola pengguna di dashboard?").
- Persyaratan unik (misalnya, "Tim kami menggunakan Paket Enterprise dan membutuhkan API untuk integrasi.").
- Konteks operasional (misalnya, "Tim kami terdiri dari 18 anggota").

Detail ini secara otomatis disimpan sebagai catatan di profil pelanggan, membuatnya mudah diakses untuk referensi di masa depan.

## Di Mana Saya Bisa Melihat Memory yang Dihasilkan Captain?

Memory yang dihasilkan Captain disimpan di bagian Notes dari dashboard pelanggan, ditautkan langsung ke pelanggan yang relevan. Ini memastikan tim Anda bisa mengaksesnya kapan saja selama percakapan.

### Cara Melihat Memories:

1. Buka profil pelanggan di dashboard.
2. Buka bagian Notes, di mana semua memory yang disimpan terorganisir dengan timestamp dan konteks untuk referensi mudah.
