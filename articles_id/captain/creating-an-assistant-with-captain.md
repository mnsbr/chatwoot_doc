
# Membuat Assistant dengan Captain

Assistant di Captain dibuat untuk membantu menjawab pertanyaan pelanggan, memberikan solusi, dan membantu masalah terkait produk. Assistant belajar dari artikel help center dan percakapan sebelumnya untuk memberikan respons yang akurat. Saat ditautkan ke inbox, assistant bisa menangani percakapan langsung dengan pelanggan Anda.

## Cara Membuat Assistant di Captain

Jika Anda menggunakan paket berbayar di Katalis.app Cloud, Anda akan menemukan menu Captain di sidebar kiri. Di bawah menu ini, Anda akan melihat opsi untuk Assistants, Documents, dan FAQs.

Klik Assistants. Anda akan melihat halaman seperti di bawah ini.

Klik tombol "Create a new Assistant".

Berikut field yang tersedia di formulir untuk membuat assistant:

| Nama Field | Deskripsi | Default | Wajib |
| -- | -- | -- | -- |
| Assistant Name | Nama internal dari assistant | -- | Ya |
| Description | Berikan deskripsi tentang apa yang dilakukan assistant. Catatan: Ini bukan instruksi untuk assistant | -- | Ya |
| Product Name | Ini penting. Assistant dibangun di sekitar produk, memiliki nama produk akan membantu mengidentifikasi kesenjangan konten dan pertanyaan dari pengguna | -- | Ya |
| Features: Enable FAQs from resolved conversations | Jika diaktifkan, Captain Assistant akan mencoba mengidentifikasi kesenjangan di artikel help center dan menyarankan FAQ baru | false | Tidak |
| Features: Capture key details as memories from customer interactions | Jika diaktifkan, akan mencoba mengidentifikasi detail penting dari percakapan dan menyimpannya di notes | false | Tidak |

Setelah memasukkan detail, klik "Create", dan assistant Anda akan ditambahkan ke akun! Anda bisa membuat beberapa assistant untuk menangani kasus penggunaan berbeda jika diperlukan.

## Cara Menghubungkan Assistant ke Inbox

Menambahkan assistant ke akun Anda tidak secara otomatis menghubungkannya ke semua inbox. Anda perlu secara manual menautkan setiap assistant ke inbox yang relevan di mana diperlukan. Ini memberikan fleksibilitas untuk menetapkan assistant berdasarkan produk tertentu atau segmen pelanggan.

Untuk menghubungkan assistant, klik menu tiga titik di samping detail assistant. Dari dropdown, pilih "View Connected Inboxes".

Ini akan membawa Anda ke halaman yang menampilkan semua inbox yang tertaut ke assistant. Dari sana, Anda bisa mengelola koneksi. Catatan: Setiap inbox hanya bisa terhubung ke satu assistant pada satu waktu.

Klik "Connect a New Inbox" untuk melihat daftar inbox yang tersedia. Pilih inbox dari daftar dan hubungkan.

Selesai! Assistant sekarang aktif di inbox. Coba kirim pesan di chat Anda dan lihat apakah ia merespons dengan sapaan awal. Anda bisa men-deploy assistant di inbox apa pun yang tersedia, baik itu live chat, WhatsApp, Instagram, email, atau lainnya.

Assistant telah berhasil disiapkan. Namun, pada tahap ini, ia belum memiliki pengetahuan tentang bisnis dan produk Anda. Setelah sapaan awal, assistant akan mencoba mentransfer percakapan ke agen. Untuk memberikan konteks tentang produk Anda ke assistant, Anda bisa menambahkan dokumen. Pelajari lebih lanjut tentang dokumen di artikel ini.
