# Cara Menyiapkan Channel SMS

Anda bisa menyiapkan inbox channel SMS melalui dua provider:

1. Twilio
2. Bandwidth

## Membuat Inbox SMS

Ikuti langkah-langkah di bawah ini.

**Langkah 1.** Buka dashboard Katalis.app Anda. Buka Settings → Inboxes → Add Inbox.

**Langkah 2.** Klik ikon "SMS".

**Langkah 3.** Pilih API provider Anda dan konfigurasikan inbox. Dijelaskan di bawah adalah kedua opsi.

### Memilih "Twilio" Sebagai API Provider

Jika Anda ingin menggunakan Twilio untuk menyiapkan inbox SMS, pilih dari dropdown. Ini akan menampilkan kolom yang diperlukan seperti yang dijelaskan di bawah.

Ini adalah input yang diperlukan untuk membuat channel ini:

Klik Create Twilio Channel setelah mengisi informasi.

#### Mengonfigurasikan Katalis.app dengan Twilio Studio

Jika Anda menggunakan Twilio Studio untuk alur percakapan kustom, memperbarui webhook URL secara langsung akan merusak integrasi yang ada.

Untuk kasus seperti itu, Anda bisa navigasi ke bagian "Twilio Studio" di dokumen ini.

### Memilih "Bandwidth" Sebagai API Provider

Jika Anda ingin menggunakan Bandwidth untuk menyiapkan inbox SMS, pilih dari dropdown. Ini akan menampilkan kolom yang diperlukan seperti yang ditunjukkan di bawah.

Klik Create Bandwidth Channel setelah mengisi informasi.

Setelah selesai dengan langkah ini, lanjutkan dengan menambahkan agen.

**Langkah 4.** "Add agents" ke inbox Anda.

Anda telah berhasil membuat inbox SMS. Anda akan mulai menerima pesan di dashboard Katalis.app Anda.

Lihat juga: Mengirim kampanye SMS ke grup kontak.
