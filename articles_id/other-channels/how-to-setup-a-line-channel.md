# Cara Menyiapkan Channel Line

**Langkah 1.** Buka Settings → Inboxes → "Add Inbox".

**Langkah 2.** Klik ikon "Line".

**Langkah 3.** Buka Line Developer Console dan buat akun Line.

**Langkah 4.** Buat "Provider" di developer console. Buat channel "Messaging API" (Bot) baru di bawah provider tersebut.

**Langkah 5.** Isi kolom di bawah dari Line developer console (Messaging API Channel) dan klik "Create LINE Channel".

1. Channel Name
2. LINE Channel ID
3. LINE Channel Secret
4. LINE Channel Token

**Langkah 6.** "Add agents" ke inbox Line Anda.

Pengaturan inbox selesai.

**Langkah 7.** Buka channel "Messaging API" di developer console dan konfigurasikan webhook.

1. Verifikasi webhook URL Katalis
2. Aktifkan "Use webhook"
