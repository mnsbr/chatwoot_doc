# Cara Menemukan Personal Access Token di Katalis

Katalis menyediakan set API gratis yang lengkap dan tersedia untuk semua pengguna. Anda bisa menjelajahi dokumentasi API lengkap kami di sini.

Untuk melakukan panggilan API ke akun Katalis, Anda memerlukan Personal Access Token untuk autentikasi. Panduan ini akan memandu Anda melalui proses menemukan access token.

**Catatan Penting:** Anda harus memiliki hak administrator pada akun untuk mengakses Personal Access Token.

## Langkah 1: Akses Profil Anda

Klik gambar avatar di pojok kiri bawah layar untuk membuka menu profil.

## Langkah 2: Buka Profile Settings

Dari menu profil, pilih "Profile Settings" untuk melihat semua opsi konfigurasi profil.

## Langkah 3: Temukan Access Token

Gulir ke bagian bawah halaman Profile Settings untuk menemukan Personal Access Token Anda.

## Menggunakan Access Token

Setelah menemukan token:

1. Salin access token
2. Sertakan dalam header permintaan API untuk autentikasi yang berhasil
3. Jaga token Anda tetap aman dan jangan pernah membagikannya secara publik

Personal Access Token berfungsi sebagai identifikasi unik Anda untuk semua interaksi API dengan akun Katalis, jadi perlakukan dengan keamanan yang sama seperti kata sandi.
