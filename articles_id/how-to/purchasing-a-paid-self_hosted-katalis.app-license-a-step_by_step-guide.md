# Panduan Langkah demi Langkah Membeli Lisensi Self-Hosted Katalis.app Berbayar

## Ikhtisar Produk Katalis.app

Katalis.app menawarkan dua opsi produk sesuai kebutuhan Anda:

1. **Katalis.app Cloud Version:** Kami menangani semua infrastruktur dan pemeliharaan sehingga Anda bisa fokus pada operasi bisnis. Cukup kunjungi katalis.app, masuk, dan Anda akan siap mengirim pesan pertama dalam waktu kurang dari 5 menit.

2. **Katalis.app Self-Hosted Version:** Opsi ini memberi Anda kontrol penuh, dengan tim Anda bertanggung jawab untuk hosting software di server sendiri dan mengelola pemeliharaan. Kami tidak memiliki akses ke instance ini, memastikan data Anda tetap sepenuhnya di lingkungan Anda.

Versi self-hosted memberikan kedaulatan data penuh sambil tetap menghadirkan semua fitur canggih Katalis.app.

Versi self-hosted kami hadir dalam dua tingkat:

- **Premium Self-Hosted Version:** Akses semua fitur PRO termasuk Captain AI dan SLA, plus menerima dukungan premium sepanjang perjalanan Katalis.app Anda.
- **Free Self-Hosted Community Edition:** Tersedia di bawah Lisensi MIT, versi ini sepenuhnya gratis untuk digunakan. Siapkan menggunakan panduan developer kami di https://developers.katalis.app/self-hosted. Perlu dicatat bahwa versi ini tidak menyertakan dukungan premium — Anda perlu menangani hosting, pemeliharaan, dan troubleshooting secara mandiri.

## Cara Membeli Lisensi Premium untuk Akun Self-Hosted Katalis.app

**LANGKAH 1:** Setelah self-hosting instance Katalis.app, akses panel Super Admin dengan mengunjungi https://<domain-anda>/super_admin dan masuk.

**LANGKAH 2:** Klik tombol settings di navigation bar kiri untuk mengakses halaman pengaturan.

**LANGKAH 3:** Di halaman settings, klik tombol "Manage" untuk membeli lisensi premium.

**LANGKAH 4:** Pilih paket harga yang paling sesuai dengan kebutuhan dan selesaikan proses pembayaran.

**LANGKAH 5:** Setelah pembayaran, kembali ke halaman settings untuk memverifikasi bahwa paket pro yang benar telah diaktifkan untuk akun Anda.

Nikmati pengalaman Katalis.app Anda! Jika ada pertanyaan, silakan hubungi tim dukungan kami. Kami dengan senang hati akan membantu Anda!
