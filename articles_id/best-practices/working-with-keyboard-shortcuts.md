# Bekerja dengan Keyboard Shortcut

Untuk meningkatkan produktivitas Anda, Anda bisa menggunakan keyboard shortcut. Shortcut ini memungkinkan Anda melakukan tindakan cepat seperti membuka percakapan, menyelesaikan percakapan, mengaktifkan sidebar, membuka Settings, dan lainnya.

Untuk menampilkan daftar keyboard shortcut yang tersedia, tekan "CMD + /" atau "Win + /" pada keyboard Anda. Atau, pilih "Keyboard Shortcuts" dari menu Profile Settings Anda.
