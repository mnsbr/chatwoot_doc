# Mengelompokkan Chat dengan Filter, Simpan sebagai Folder

Anda bisa mengorganisir dan mengurutkan percakapan menggunakan filter lanjutan dan menyimpannya sebagai folder. Folder-folder ini muncul di sidebar untuk akses cepat ke percakapan tertentu dalam tampilan pengguna.

Untuk menggunakan fitur ini, ikuti langkah-langkah berikut.

**Langkah 1.** Klik ikon filter di bagian atas daftar pesan.

**Langkah 2.** Modal akan terbuka, yang memungkinkan Anda memilih jenis filter, operator (equal to, not equal to, present, not present) dan nilainya. Pilih dan terapkan filter Anda.

**Langkah 3.** Setelah menerapkan filter, Anda akan melihat semua percakapan yang memenuhi syarat. Selanjutnya, klik tombol "Save" di bagian atas panel daftar percakapan.

**Langkah 4.** Modal akan terbuka meminta Anda memberi nama filter.

## Cara Menghapus Folder

Untuk menghapus folder, klik ikon tempat sampah merah, seperti yang ditunjukkan pada screenshot berikut. Untuk mengonfirmasi, klik tombol "Yes, Delete".
